# Frontend

React + Vite dashboard for the distributed job scheduler.

Important directories:

- `src/app/router`: application routes
- `src/api`: Axios API clients
- `src/components`: reusable layout and UI components
- `src/features`: feature screens
- `src/lib`: browser utilities such as auth token and Socket.IO client setup
- `src/styles`: Tailwind entrypoint

Phase 1 provides the operational dashboard shell and route map. Full CRUD screens are implemented in later phases.
